# template 
スライドのテンプレート集です。

## 使い方
### title.md
タイトルスライドのテンプレートです。

### simple.md
シンプルな箇条書きスライドのテンプレートです。

### card-list.md
縦に3つのカード型リストを表示するスライドのテンプレートです。

カードにはタイトル歯科含められないので、目次など全体の概要を示すのに適しています。

### gird-2*1.md
2列のグリッド型リストを表示するスライドのテンプレートです。

### gird-3*1.md
3列のグリッド型リストを表示するスライドのテンプレートです。

## grid-2*2.md
2行2列のグリッド型リストを表示するスライドのテンプレートです。
