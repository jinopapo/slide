# template 
スライドのテンプレート集です。

## 使い方
### title.md
タイトルスライドのテンプレートです。

### simple-list.md
シンプルな箇条書きスライドのテンプレートです。

### code-block.md
タイトルと複数行のコードブロックを表示するスライドのテンプレートです。
コード例の解説やサンプル実装を見せる用途に適しています。

### card-list.md
縦に3つのカード型リストを表示するスライドのテンプレートです。

カードにはタイトルしか含められないので、目次など全体の概要を示すのに適しています。


### card-row-center-3.md
横に3つのカードを並べ、スライドの上下左右中央に配置するテンプレートです。
cardとcardの間には矢印が入るので、プロセスやフローを示すのに適しています。
各カードには単語のような短い文章と1,2つの短い箇条書きが含められます。

### card-row-center-4.md
4つのカードを並べ、スライドの上下左右中央に配置するテンプレートです。
cardとcardの間には矢印が入るので、プロセスやフローを示すのに適しています。
各カードには単語のような短い文章と1,2つの短い箇条書きが含められます。


### split-bottom-list-top-cards.md
スライドを上下で2分割し、

- 上：横に3〜4枚のカードを上下左右中央揃えで表示
- 下：箇条書き

を行うテンプレートです（3枚なら `grid-3`、4枚なら `grid-4` を利用）。

上部のカードは単語のような短い文章+1,2つの短い箇条書きしか表示できないので、下部の箇条書きで説明を補足する形になります。
また、下部の箇条書きは最大5つまでに抑えるのが見やすいです。


### gird-2*1.md
2列のグリッド型リストを表示するスライドのテンプレートです。
タイトルには短い単語やフレーズを使い、必要に応じてリストアイテムで補足説明を加える形が適しています。

### gird-3*1.md
3列のグリッド型リストを表示するスライドのテンプレートです。
タイトルには短い単語やフレーズを使い、必要に応じてリストアイテムで補足説明を加える形が適しています。


### gird-2*2.md
2行2列のグリッド型リストを表示するスライドのテンプレートです。
タイトルには短い単語やフレーズを使い、必要に応じてリストアイテムで補足説明を加える形が適しています。
